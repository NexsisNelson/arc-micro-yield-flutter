# Arc Micro-Yield Flutter App

A modern Flutter application for the Arc Micro-Yield DeFi protocol.

## Features

- Multi-Platform Support (iOS, Android, Web, Windows, macOS, Linux)
- Web3 blockchain integration with web3dart
- Deposit/Withdraw USDC
- Portfolio tracking and yield visualization

## Quick Start

```bash
flutter pub get
flutter run
```

## Development

```bash
# Build
flutter build apk --release
flutter build ios --release
flutter build web --release

# Test
flutter test
flutter analyze
```

## Project Structure

- `lib/` - Dart source code
- `test/` - Test files
- `android/`, `ios/`, `web/`, `windows/`, `macos/`, `linux/` - Platform-specific code

## Dependencies

- flutter
- cupertino_icons (v1.0.8)
- web3dart (v3.0.2)
